BATTLE PANIC - FREE GAME ASSETS GUIDE
======================================

This game uses procedural (code-generated) graphics by default.
To use real sprite graphics, download free assets and place them in these folders.

SPRITESHEET FORMAT:
-------------------
Spritesheets should be horizontal strips with each frame side by side.
Example: A 4-frame 64x64 walk animation = 256x64 PNG

Frame layout: [Frame1][Frame2][Frame3][Frame4]


TIPS:
-----
1. The game will automatically use procedural graphics if assets aren't found
2. You can replace assets one at a time - missing assets use procedural fallback
3. For best results, use pixel art with similar style across all assets
4. PNG format with transparency recommended
5. Keep file sizes small for faster loading


QUICK START:
------------
1. Go to https://itch.io/game-assets/free/tag-medieval
2. Download a medieval/fantasy character pack
3. Rename and organize sprites into the folders above
4. Refresh the game - new sprites will be loaded automatically!
